# Daily Expenditure Calculator

A simple console-based application written in **C** to help users calculate and manage their daily expenses. This project is designed to make personal expense tracking easier through structured input, logical calculation, and clear output in the terminal.

---

## Overview

The **Daily Expenditure Calculator** allows users to enter their daily spending amounts and calculates the total expenditure. It is a lightweight and practical project that demonstrates core programming concepts in C such as variables, loops, conditions, functions, and input/output handling.

This project is suitable for beginners and showcases problem-solving through a real-life use case.

---

## Features

- Add daily expense amounts
- Calculate total daily expenditure
- Simple console-based interface
- Easy user input and output handling
- Lightweight and fast execution
- Built using structured programming concepts

---

## Technologies Used

- **C Programming Language**

- Variables and data types
- Conditional statements

- Structured programming
- Basic calculation logic

---

## Project Objective

The main objective of this project is to create a simple utility program that helps users track their daily expenses and calculate total spending efficiently. It also serves as a practical exercise for strengthening fundamental C programming skills.

---

## How It Works

1. The user runs the program
2. The program asks for daily expense inputs
3. The user enters expense values
4. The program calculates the total expenditure
5. The result is displayed in the console

---

## Example Use Case

A user wants to calculate how much money they spent in a day on food, transport, shopping, and other expenses. By entering each expense into the program, the user can quickly see the total daily expenditure.
